# 📡 Wi-Fi Analyzer

A lightweight command-line Wi-Fi Analyzer tool built with Python.
Scans nearby wireless networks and displays detailed information
including signal strength, security type, BSSID, and channel.

## Features
- 🔍 Scans all nearby Wi-Fi networks instantly
- 📊 Visual signal strength bar with percentage
- 🔒 Shows security type (WPA2, Open, etc.)
- 📡 Displays channel and BSSID (MAC address)
- 🟢 Color-coded signal quality ratings
- 🔄 Live refresh support
- 💻 Cross-platform: Windows, Linux & macOS

## Requirements
- Python 3.x
- No extra libraries needed (uses built-in system commands)

## Usage
# Windows
python wifi_analyzer.py

# Linux
sudo python3 wifi_analyzer.py

# macOS
python3 wifi_analyzer.py
